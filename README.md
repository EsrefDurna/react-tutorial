# react-tutorial

## 安装

```
  # 安装依赖
  $ npm install

```

**有以下两种方法运行项目**

### 1. 不采用devServer运行项目

```bash
  # 编译代码
  $ npm build

  # 查看效果
  $ npm start

```
### 2. 采用devServer运行项目

```bash
  # 采用webpack-dev-server编译代码
  $ npm start

```
